# CandyMo Coffee

CandyMo Coffee'nin resmi web sitesi.

## 🚀 Özellikler

- Responsive tasarım
- Menü sayfaları
- Özel içecekler
- Sıcak ve soğuk içecekler
- Tatlılar ve atıştırmalıklar

## 🛠️ Teknolojiler

- HTML5
- CSS3
- JavaScript
- Netlify (Hosting)

## 📱 Menü Kategorileri

- ☕ Sıcak Kahveler
- 🧊 Soğuk Kahveler
- ☕ Türk Kahvesi, Sıcak Çikolata ve Salep
- 🍵 Çay ve Bitki Çayları
- 🥤 Meşrubatlar
- 🧋 Bubble
- 🥛 Milkshake
- ❄️ Frozen
- 🍹 Frappe
- 🍸 Mokteyl
- 🍵 Matcha
- 🍰 Tatlılar
- 🍪 Atıştırmalıklar
